DVWA 渗透教程 新手 学习笔记

0x00 关于DVWA

POINT 1. 啥是DVWA？

DVWA 是一种基于PHP+MYSQL的渗透环境，从鉴定PHP+MYSQL的Web应用的安全程度来帮助安全工作人员更好的学习和练习渗透技术。

DVWA一共有分为十个模块，每个模块四种类别，low, medium, high , impossible, 具体如下：

Brute force 暴力破解

Command injection 常规注入

CSRF 跨站请求伪造

File inclusion 文件包含

File upload 文件上传

Insecure captchar 不安全验证码

SQL injection SQL注入

SQL injection(blind) SQL注入之盲注

XSS (reflected) 反射型XSS

XSS (stored) 存储型XSS
